Search anime, characters, profiles and more from the MyAnimeList website!



Slash Command Description
✅ help Sends the link to this website.
❌ profile Sends data on your MyAnimeList profile.
✅ search-anime Search through MyAnimeList's anime database.
❌ search-manga Search through MyAnimeList's manga database.
✅ search-character Search through MyAnimeList's character database.
❌ search-profile Search through MyAnimeList's profile database.
❌ recommend-anime Recommands anime from MyAnimeList's anime database.
❌ recommend-manga Recommands manga from MyAnimeList's manga database.
❌ top-anime Current top animes from MyAnimeList's anime database.
❌ top-manga Current top mangas from MyAnimeList's manga database.
❌ top-character Current characters manga from MyAnimeList's character database.
❌ season Current season of anime from MyAnimeList's anime database.
❌ upcoming-seasom Upcoming season of anime from MyAnimeList's anime database.
❌ random-anime Gets a random anime from MyAnimeList's anime database.
❌ random-manga Gets a random manga from MyAnimeList's manga database.
❌ random-character Gets a character manga from MyAnimeList's character database.
